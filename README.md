# JavaTraining-Week3

タスク管理アプリを作成しよう

## アプリの概要

コマンドライン上で操作できる、シンプルなタスク管理アプリです。  
タスクの登録・一覧表示・完了・削除ができ、終了時には登録件数と完了済み件数を表示します。

## 実行方法

1.cd src

2.javac AppMain.java

3.java AppMain

## 工夫した点、苦労した点

・クラスの役割分担をするうえでどこ何を書くのが自然かを考えるのに悩みました。必要な要件を列挙しただけでそのまま進めるのではなく、各クラスにどの役割が必要か、必要のない役割はないかを考えながら進めたのが工夫した点です。

・タスク一覧の表示では、forEach とラムダ式を使い、処理の流れが分かりやすくなるように工夫しました。初めに for 文で実装してから、処理の流れを理解したうえで置き換えました。

・入力時の例外処理については、どこまでチェックするかの線引きに迷いましたが、今回はタイトルや期限が空文字の場合の対応に絞っています。

・全体として、課題要件を満たすことを優先しつつ、処理を複雑にしすぎないようバランスを意識して実装しました。

## 応用課題

・Stream API を用いて、完了済みタスク数を集計・表示する処理を実装しました。  
forEach やラムダ式とあわせて、コレクションを扱う処理の書き方を意識しています。
